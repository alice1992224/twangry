<h1 id="">錯誤處理</h1>

<h3 id="">有政誌上的事件內容不大對勁，我應該如何處理？</h3>

<ol>
<li>每個事件上方有「編輯內容」，連結至對應的維基百科條目。</li>
<li>在維基百科中，你可以選擇要編輯的段落，直接在段落標題旁按下編輯。</li>
<li>編輯條目內容，需要標注具體的事實或內容來源，請尊重維基百科的規範，附上實際來源，在政誌中也會以畫面自動截圖呈現。</li>
<li>確認維基百科內容更新後，可至政誌的事件時間軸最下方，按下「<img src="http://fact.g0v.tw/css/images/reload.gif" alt="更新" title="">」符號，政誌頁面會自動更新至最新的維基百科版本</li>
</ol>

<h3 id="">發現內容時間有錯，應該如何處理？</h3>

<p>政誌應用程式自動判斷時間，但仍然有機會找錯年份，此時我們需要你的指正，若你能直接改正就更棒了，我們建議：</p>

<ol>
<li>幫忙核對是否原始資料在維基百科有誤，若有錯誤，直接編輯該段落，改正日期時間即可。</li>
<li>若原始資料是正確的，但政誌錯誤，請張貼錯誤連結至<a href="https://github.com/g0v/twangry/issues/new" target="_blank">Github</a>，標示正確的日期，讓我們趕緊處理！</li>
</ol>

<h3 id="">程式錯誤回報</h3>

<ul>
<li>請用 github 的 <a href="https://github.com/g0v/twangry/issues" target="_blank">issue report</a></li>
</ul>
